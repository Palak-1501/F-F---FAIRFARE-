# FAIRFARE - Eco-Friendly Fare Intelligence Platform 🚖

## 📌 Project Overview
FAIRFARE is a web-based platform that helps users compare ride fares across multiple ride-hailing services like Uber, Ola, Rapido, etc. It provides price insights and eco-friendly ride options to help users make smarter decisions.

---

## 🎯 Objective
To create a demo web platform that:
- Compares fares across multiple platforms
- Provides estimated prices and ETAs
- Suggests eco-friendly ride options (EV vehicles)
- Simulates booking and payment flow

---

## 🛠 Tools & Technologies Used
- HTML, CSS, JavaScript (Frontend)
- SQL (Database Design)
- Local Storage (for demo data handling)
- UI/UX Design (Interactive Web Interface)

---

## ⚙️ Features
- User Login & Signup system
- Location selection (State & City)
- Ride fare comparison (Uber, Ola, etc.)
- Vehicle selection (Bike, Auto, Sedan, EV)
- Price estimation & ETA calculation
- Booking confirmation system
- Payment simulation (Card, UPI, QR)
- Booking history tracking

---

## 🗂 Project Files
- `index.html` → Complete frontend web application
- `DATABASE.sql` → Database schema (users, bookings, payments, etc.)

---

## 📊 Database Design
Includes tables for:
- Users
- Bookings
- Drivers
- Platforms
- Payments

---

## 🚀 How to Run the Project
1. Open `index.html` in any browser  
2. Explore the platform features  
3. (Optional) Run `DATABASE.sql` in MySQL to create database  

---
